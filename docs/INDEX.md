本リポジトリの Document System の運用方針は [document_system.md](./document_system.md) を参照。

## カテゴリ

- [`business/`](./business/INDEX.md): プロダクト概要・利用者像などのビジネス文書
- [`development/`](./development/INDEX.md): 技術スタック・アーキテクチャ・データモデル・インフラなどの開発文書
- [`operations/`](./operations/INDEX.md): デプロイ・監視など運用文書（今後整備）
