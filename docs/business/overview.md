# Cheatdex プロダクト概要

## 一行説明

ゲームエミュレータのチートコードを、プラットフォーム・ゲーム単位で整理・管理し、エミュレータが読み込める形式でまとめて書き出せる Web アプリケーション。

## 解決する課題

エミュレータのチートコードは、

- プラットフォーム（PS2 / PS / SNES など）ごとにファイル形式が異なる
- ゲームごとにコードが散在し、どれを有効化しているか管理しづらい
- 複数の端末・エミュレータ間で設定を持ち回りにくい

Cheatdex はこれらを 1 か所に集約し、必要なチートだけを ON にして、エミュレータ向けの zip にまとめて取り出せるようにする。

## コア機能

| 機能 | 説明 |
|------|------|
| プラットフォーム管理 | 固定マスタとして提供されるプラットフォーム単位でチートを分類（MVP は PS2 のみ） |
| ゲーム管理 | プラットフォーム配下にゲームを登録し、ゲーム単位でチートを束ねる |
| チートコード管理 | ゲームに紐づくチートコードを登録・編集 |
| Chrome 拡張による収集 | 開いているページから既知形式（GameShark 系 hex / PCSX2 `.pnach`）のチートコードを正規表現で検出し、まとめて Cheatdex に登録 |
| ON/OFF 切替 | 各チートコードの有効/無効を切り替え |
| zip ダウンロード | 全チート、またはプラットフォーム単位で zip 一括ダウンロード |
| Google ログイン | Web ログイン用に Google アカウントによる認証 |
| 拡張用トークン発行 | Chrome 拡張から API を叩くための PAT（Personal Access Token）を Web 上で発行 |

## スコープ

### 含む（MVP）

- 上記コア機能（PS2 / PCSX2 `.pnach` 形式での zip 出力を含む）
- Chrome 拡張による収集（GameShark 系 hex / `.pnach` の 2 形式に限定）

### 含まない（現時点）

- PS2 以外のプラットフォームへの対応（MVP 後に拡張）
- チートコードのインポート（既存ファイルからの取り込み）／外部チートDBとの連携
- チートコードの自動検証・動作保証
- ユーザー間でのチート共有・公開
- Chrome 以外のブラウザ拡張（Firefox / Edge） / Android アプリ

> 上記スコープは要件確定後に見直す。YAGNI に従い、顕在化していない要件は実装しない。
